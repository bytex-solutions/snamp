SNAMP ${project.version}

** How to run **
Default start process (called Regular Mode):
1. Open command-line shell
2. cd <snamp-installation-directory>
3. Linux: sh bin/karaf. Windows: bin\karaf.bat
4. Print snamp: in the command-line shell. Then press <TAB> and you will see the list of all available SNAMP commands
5. Check Web Console at http://localhost:3535/snamp
