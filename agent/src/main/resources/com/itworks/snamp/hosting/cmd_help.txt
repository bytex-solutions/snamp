Usage: java snamp <config-file> [config-format]

Additional JVM properties used by SNAMP:
    -Dcom.snamp.plugindir Path to the directory with SNAMP plugins, by default "./plugins" is used
    -Dcom.snamp.manager The name of the SNAMP manager plugin. This plugin should be available
via directory with plugins or Java Class Path. By default, SNAMP uses command line manager
    -Djava.util.logging.config.file Path to the logging configuration file

Examples:
    java snamp config.yml yaml
    java snamp config.yml
    java -Dcom.snamp.plugindir /usr/bin/snamp/addons snamp config.yml yaml
